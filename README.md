# Aspartam Frontend

![Build status](https://circleci.com/gh/hawkaa/aspartam-frontend.svg?style=shield&circle-token=:circle-token)

#### Getting started
```
yarn
```

#### Development server
```
yarn start
```

#### Running tests
```
yarn test
```

#### Linting
```
yarn lint
```

#### Building for production
```
yarn build
```
